## 🌱 Intereses

### Sistemas embebidos e IoT

Diseño de soluciones que integren microcontroladores, sensores, actuadores, comunicación serial y lógica de control para monitoreo o automatización de procesos físicos.

### Desarrollo web y automatización

Construcción de plataformas web funcionales, sistemas de consulta, generación de documentos, formularios, reportes y trazabilidad de información.

### Diseño electrónico y PCB

Diseño de circuitos, selección de componentes, validación de footprints, etapas de potencia, protección eléctrica, montaje SMD y prototipado de hardware real.

### Inteligencia artificial embebida

Aplicación de modelos livianos para interpretar datos sensóricos, generar alertas, clasificar estados y apoyar decisiones directamente en dispositivos pequeños.

### Visualización técnica

Uso de Python, Matplotlib, NumPy, Pandas y Manim para convertir datos experimentales o simulados en gráficas claras, explicativas y útiles para tomar decisiones técnicas.

### Sistemas digitales

Programación y análisis de sistemas digitales con VHDL, Verilog, Assembly, PIC18F4550 y flujos de compilación, síntesis, generación de bitstreams y programación de hardware.
